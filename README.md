**A Tinder-like dating app using the MVVM architecture**

**Features**:
- Swiping functionality
- Card view
- Like/Dislike users
- Matching users
- Upload upto 3 photos
- Changing photos
- Creating an account
- Storing user's info in Firestore
- Age filtering
